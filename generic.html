AI Resume Reviewer

An AI-powered application that analyzes resumes and provides structured feedback on formatting, content, and industry-specific improvements. Implemented resume text extraction using PyPDF2 and io for PDF parsing, integrated the OpenAI API for natural language processing and feedback generation, and utilized Streamlit for an interactive front-end interface. Leveraged dotenv for secure environment variable management and optimized file handling for both PDF and TXT formats.
